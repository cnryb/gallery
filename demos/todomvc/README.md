# TodoMVC Demo

## Getting Started

1. Install dependencies
    ```shell
    $ npm install
    ```

2. Start the development server
    ```shell
    $ npm start
    ```

3. Preview in kraken
    ```shell
    $ kraken http://localhost:9999/kraken/index.js
    ```

4. Preview in web browser
    ```shell
    $ open http://localhost:9999/
    ```
